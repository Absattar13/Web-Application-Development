Dice Game
Nuridin Absattar
Instructions
1. Open `index.html` in a web browser.
2. Click the "Roll Dice" button to roll the dice.
3. The result of each roll is displayed, along with the sum and roll count.
Additional Features
- Displays the sum of the two dice.
- Roll counter keeps track of the number of rolls.
- A special message is shown when doubles are rolled.
